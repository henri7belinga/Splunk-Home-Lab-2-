<h1>Analyzing HTTP Log Files Using Splunk SIEM</h1>


<h2>Description</h2>

In this project, I utilized Splunk Enterprise to perform an in-depth analysis of HTTP logs. I started by uploading a compressed HTTP log file into Splunk and executed a detailed triage process to prioritize key areas of concern. Through advanced field extraction, I isolated critical HTTP data points, which enabled a comprehensive analysis of web traffic patterns. I focused on detecting anomalies, identifying suspicious activities, and monitoring user behavior to assess potential security risks. The project highlights my expertise in leveraging Splunkâ€™s capabilities for data-driven insights and enhancing network security by proactively identifying and addressing threats.
